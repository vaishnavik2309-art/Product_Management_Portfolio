# AI-Powered Meeting Notes Automation

## Overview
Built an AI automation workflow that automatically converts Google Calendar meeting notes into structured summaries, action items, and shareable documentation.

## Problem Solved
Meeting discussions often get lost after calls, making it difficult to track decisions, action items, and ownership. Teams spend significant time manually documenting meeting outcomes. :contentReference[oaicite:1]{index=1}

## Solution
Created an end-to-end workflow that:
- Detects completed meetings from Google Calendar
- Extracts meeting notes automatically
- Uses AI to generate structured summaries
- Creates a Google Doc for record keeping
- Sends summary emails to stakeholders

## Workflow
Google Calendar → Make → OpenAI → Google Docs → Gmail :contentReference[oaicite:2]{index=2}

## Key Features
- Automatic meeting summarization
- Action item extraction
- Owner and deadline identification
- Structured documentation generation
- Email-based summary distribution
- Searchable knowledge repository
