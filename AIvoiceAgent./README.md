# AI Voice Agent for Restaurant Reservations

## Overview
Built an AI voice agent using ElevenLabs that handles restaurant reservation calls, collects customer details, answers booking-related queries, and confirms reservations through natural conversations.

## Key Features
- Natural voice-based customer interactions
- Automated table reservation booking
- Customer detail collection and validation
- Reservation confirmation and follow-up responses
- 24/7 availability without human intervention

## User Flow
Customer calls → AI greets customer → Collects reservation details (date, time, guests) → Confirms availability → Shares booking confirmation.

## Tools & Technologies
- ElevenLabs Conversational AI
- Prompt Engineering
- Voice Agent Configuration
- Conversation Flow Design

## Outcome
Demonstrated how AI voice agents can automate routine customer interactions, reduce operational workload, and improve customer experience for restaurants.
