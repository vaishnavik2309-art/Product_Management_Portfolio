# Football Dataset Analysis using SQL

## Overview
Analyzed a football player, team, and transfer dataset using SQL to uncover performance insights, player efficiency metrics, transfer patterns, and team performance trends.

## What I Did
- Queried player performance data
- Identified top goal scorers
- Evaluated player value for money based on salary and goals scored
- Analyzed team performance and league standings
- Examined player transfer activities
- Identified potential underperforming players
- Measured team budget efficiency

## Key Insights
- Identified the top-performing goal scorers across teams.
- Found players delivering the highest value relative to salary.
- Compared team success against financial investment.
- Analyzed transfer movements between clubs.
- Flagged players with low output despite high playtime.
- Evaluated which teams achieved the best results with available budgets.
